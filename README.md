# BTS.id Backend Test

## Requirement

1. Node JS (v16.17.0+)
2. Docker (v4.12.0)

## Installation

Install terlebih dahulu dependencies
```bash
npm install
```

Buat database dengan menggunakan **Docker**
```bash
docker compose up -d
```

Setelah database dibuat, jalankan aplikasi
```bash
npm start
```

## API Collection

File collection [bts-collection.json](bts-collection.json) untuk melakukan testing pada software **Postman**
